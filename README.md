# TomorrowRich Backend

내일은 부자 서비스의 Spring Boot 백엔드 프로젝트입니다.

## Tech Stack

- Java 17
- Spring Boot
- Gradle
- Spring Data JPA
- Spring Security
- H2 Database
- JWT

## Run

```bash
./gradlew bootRun
```

## Test

```bash
./gradlew test
```

## Branch Rule

- `main`: stable branch
- `develop`: development integration branch
- `feature/*`: feature work branch
